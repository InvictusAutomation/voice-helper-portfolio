## 🔧 关键技术要点

### 1. 音画同步
所有字幕的出现时机必须**严格跟随语音节奏**。建议：
1. 先把所有音频按顺序铺在时间轴上
2. 听一遍，标记每个句子起止点（打标记点）
3. 根据标记点放置对应视频和字幕

### 2. 色彩过渡
整个视频的色彩走向：
```
海岛蓝(87CEEB) → 梦幻紫(1A1A2E+DDA0DD) → 夏日明(4FC3F7) → 科技青(0D9488→0A1628) → 暖白(FFF8E7) → 极简黑(0A1628) → 纸张白(F5F0E8)
```

### 3. 出镜视频处理
你的三段出镜视频 (`1-1`, `1-2`, `2-1`) 建议：
- **统一样式**: 居中偏下、圆角、外阴影
- **统一大小**: 约1100-1200px宽
- **边缘处理**: 可加轻微 vignette（暗角），聚焦视觉中心
- 如果原视频有杂音/环境音：降低原声音量到30%，后期配音轨为主

### 4. GIF处理 (`phineas-ferb-giphy.gif`)
- 剪映: 直接导入会自动循环播放
- PR: 导入为图片序列或保持GIF格式
- 注意: 必须保留版权标注

### 5. 书写笔记照片 (`IMG_4016/7/8`)
- 这些是手写研究笔记，展示你"在探索AI"
- 切换时用**交叉溶解**（Cross Dissolve），不要硬切
- 可以加极轻微的Ken Burns效果（缓慢推近/平移）

## 📝 剪映操作步骤摘要

### 第一步：新建项目
1. 打开剪映 → 新建项目 → 1920×1080, 30fps

### 第二步：导入素材
导入以下文件夹中的全部素材：
```
docs/video-prep/clips/     (所有视频和MP3)
docs/video-prep/assets/   (tw.mp4, GIF, JPEG/PNG, lottie/, icons/)
```

### 第三步：按Shot顺序铺设
按照上面的「镜头分解表」，逐个Shot铺设素材：
1. 先铺音频轨（确定节奏）
2. 再铺视频轨（对齐音频）
3. 最后加字幕和特效

### 第四步：精调
1. 逐帧检查音画同步点
2. 调整每个转场的时长和曲线
3. 检查字幕可读性
4. 确认版权标注已添加

### 第五步：导出
- 格式: MP4
- 编码: H.264
- 码率: 10-15 Mbps
- 音频: AAC, 320kbps
