## 需要你提供的素材

### 📷 照片类（可选）
1. **个人照片** - 侧脸或背影，不要正面直视镜头
2. **画画场景** - 如果你有画画的场景照片
3. **新加坡时期** - 走廊、教室或任何有回忆的照片

- "在新加坡，我在走廊上抓人办读书会——用英文给十个国家的人，讲我对AI的观察"
- "2026，AI Agent兴起——我看到人们深入思考、丰富体验的可能"
- "Don't be the best. Be the only. 我想探索：人机交互下，人类思维的变迁"
